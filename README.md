
# Syntecxhub Landing Page Project

This project is a responsive Landing Page website developed as part of my Front-End Development Internship at Syntecxhub.

The objective of this project was to design and build a clean, modern and fully responsive landing page using HTML, CSS and JavaScript.

---

## 🚀 Project Overview

The landing page represents a digital product/service website and includes:

- Responsive Navigation Bar
- Hero Section
- Features Section
- Call-to-Action Section
- Footer

The design focuses on simplicity, readability and modern UI practices.

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Media Queries)
- JavaScript (for mobile menu toggle)

No external frameworks or libraries were used.

---

## 📱 Features

- Fully Responsive Design (Mobile, Tablet, Desktop)
- Smooth Scrolling Navigation
- Mobile Hamburger Menu
- Hover Animations on Buttons & Cards
- Clean Layout using Flexbox
- Modern Color Scheme
